Read Me

This is the user manual for ABC Book Store Web API.
Type in the follow URLs to get results.

http://localhost:8080/BookStoreWebAPI/services/help

Returns help page.

http://localhost:8080/BookStoreWebAPI/services/saveInvoices

Save the invoices in the database.

http://localhost:8080/BookStoreWebAPI/services/version

Returns version number of the API.

http://localhost:8080/BookStoreWebAPI/services/status

Returns the API status.

http://localhost:8080/BookStoreWebAPI/services/status/getRecommends/{cutomer id}

Returns the recommended books for a given customer. Customer id should be provided in the url.
